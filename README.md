# 🛍️ Shopify App

A modern, full-featured **E-Commerce** mobile application built using **Flutter**. The app follows the **MVVM architecture**, integrates with **RESTful APIs**, and includes essential shopping features with a clean and professional UI design.

---

## ✨ Features

- 🔍 **Search Products**: Quickly find products using a responsive search interface.
- 🛒 **Add to Cart**: Add items to your cart and manage them easily.
- ❤️ **Liked Products**: Save and view your favorite products.
- 👤 **User Profile**: Each user has a dedicated profile page.
- 🔄 **State Management**: Efficient and clean state handling for UI and business logic with bloc state management.
- 🌐 **RESTful API Integration**: Connects to live backend services for product data, user actions, and more.
- 🎨 **Professional UI Design**: Smooth and modern design focusing on user experience.

---

## 🧱 Tech Stack

- **Flutter**
- **Dart**
- **MVVM**
- **Bloc**
- **RESTful APIs** for backend communication
